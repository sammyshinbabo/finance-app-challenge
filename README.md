# My Spending Dashboard
## Overview
This is a simple web dashboard to track your money. It helps you see your income and expenses clearly. The project gets starting data from a mock API. You can add new transaction, search through them, and see your total balance.

## Features
View Transactions: See a list of your money records in a clean table.

* Pagination: Shows 5 items per page to keep the screen looking neat.

* Search and Filter: Search for a specific record by typing, or choose a category from the dropdown menu.
 
* Total Calculation: It adds your income and subtracts your expenses to show your current money balance.

* Add New Transaction: Click the Add button to create a new transaction. You can choose if it is an income (+) or an expense (-).

## Technologies Used
* HTML: For the structure of the page.

* CSS: For the design, colors, and layout.

* JavaScript: To make the buttons work, calculate the math, and get data from the API.

## File Structure
* index.html: The main web page file.

* style.css: The design file.

* script.js: The logic and math file.

## How to Setup and Run
### Using VS Code
* Open the project folder in VS Code.

* Go to the Extensions tab and install Live Server.

* Right-click on the index.html file and click Open with Live Server.

* Your web browser will open the dashboard.
